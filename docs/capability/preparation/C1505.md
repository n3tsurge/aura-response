# Establish Ability to List Registry Keys Created
&nbsp;![](https://img.shields.io/badge/ID-C1505-blue)&nbsp;![](https://img.shields.io/badge/Phase-Preparation_%28P0001%29-blue)&nbsp;![](https://img.shields.io/badge/Category-Configuration-blue)
## Overview
This capability establishes the ability to list registry keys that have been created on a system. This is essential for monitoring changes to the registry, which can indicate potential security incidents or configuration changes.

## Tools
The following tools provide this capability:

- [Microsoft Sentinel](../tool/ms-sentinel/C1505.md)
- [SentinelOne](../tool/sentinelone/C1505.md)
