# Establish Ability to Quarantine File by Path
&nbsp;![](https://img.shields.io/badge/ID-C1312-blue)&nbsp;![](https://img.shields.io/badge/Phase-Preparation_%28P0001%29-blue)&nbsp;![](https://img.shields.io/badge/Category-File-blue)
## Overview
This capability involves establishing the ability to quarantine a file by its path within a system or network. This includes implementing mechanisms to isolate files based on their directory structure and file names, ensuring that potentially harmful files can be effectively contained and managed. It is essential for maintaining data integrity, preventing the spread of malware, and enhancing overall security posture by providing a controlled environment for suspicious files.
